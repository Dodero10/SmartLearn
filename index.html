Tính năng

Giao diện hiện đại

Giao diện người dùng trực quan, thân thiện với trải nghiệm mượt mà và thiết kế responsive.

Backend mạnh mẽ

Xây dựng trên nền tảng FastAPI với Python 3.11, đảm bảo hiệu suất cao và khả năng mở rộng.

Docker tích hợp

Triển khai dễ dàng với Docker, giúp dự án của bạn chạy nhất quán trên mọi môi trường.

Cài đặt

1. Chạy Frontend


# Tạo file .env từ file .env.example
cp .env.example .env

# CD vào folder chatbot-frontend
cd chatbot-frontend

# Install dependencies
npm install

# Run frontend
npm start

2. Chạy Backend


# Tạo file .env từ file .env.example
cp .env.example .env

# Run docker compose
docker compose up --build
# hoặc
docker compose up -d

# CD vào folder chatbot-backend
cd chatbot-backend

# Tạo môi trường với anaconda
conda create -n myenv python=3.11

# Activate môi trường
conda activate myenv

# Install dependencies
pip install -r requirements.txt

# Seed data vào database
python app/database/seed.py

# Run backend
uvicorn main:app --reload --host 0.0.0.0 --port 8030
